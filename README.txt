Must import the algs4.jar file into the project to run the Solver.java program as is needed for the imports.

Puzzle.java is neeeded in the project to run the Solver.java program.

In order to switch between 8-puzzle and 15-puzzle, include both .txt files into the project.

Edit the "In in = new In("15puzzle.txt");" to "In in = new In("8puzzle.txt");" 
in both the Board.java and Solver.java to switch from the 15-puzzle to the 8-puzzle.
